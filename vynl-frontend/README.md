A minimal, futuristic but warm React + Tailwind + Framer Motion homepage inspired by SoundCloud's layout.

## Quick start

```powershell
# from the vnyl_app folder
npm install
npm run dev
```
Then open the printed local URL (defaults to http://localhost:5173).

## Build and preview
```powershell
npm run build
npm run preview
```

## Stack
- Vite + React + TypeScript
- TailwindCSS (utility-first styling)
- Framer Motion (animations)
- Google Fonts: Inter, Space Grotesk

## Notes
- Color palette: off-black (#0e0e0e), soft white (#f5f5f5), accent orange (#FF6B00)
- Responsive, accessible buttons and focus states
- No backend logic yet; UI only
